## ğŸ“¥ Installation

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git installed on your system
- Text editor (VS Code, Sublime Text, etc.) - optional

### Clone the Repository

```bash
git clone https://github.com/msneeraj928-tech/WebTech_Practical_Assignment.git
cd WebTech_Practical_Assignment
```

### Running Experiments

Simply open any experiment HTML file in your browser:

```bash
# On Windows
start 1.html

# On macOS
open 1.html

# On Linux
xdg-open 1.html
```

### Running Portfolio Locally

Use a local server for the best experience:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Then visit http://localhost:8000
```
